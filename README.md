 # Третье задание: Таск-менеджер для команды

## Инструкция по запуску проекта

1. Клонировать репозиторий:
git clone https://github.com/DmitriiTMS/MLE-TASK-3.git

2. Перейти в директорию проекта:
cd MLE-TASK-3

3. Настроить переменные окружения:
cоздайте файл .env, скопировав содержимое .env.example, и установите необходимые значения для переменных.

4. Запустить проект с помощью Docker:
docker compose up --build

5. Тестовые данные:
gри первом запуске проекта автоматически создаются тестовые данные в базе данных.
На данном этапе доступны GET-запросы, которые не требуют авторизации.

6. Импортировать коллекцию Postman:
bмпортируйте файл MLE-TASK-3.postman_collection из корня репозитория в Postman.

7. Готово!

Запуск тестов: npm run test
